# Fleet wiki vault

Obsidian-style markdown: use **`[[wikilinks]]`** between notes, keep **one canonical note per fact**, and link from **`00-MOC-Fleet.md`**.

- **`fleet/`** — fleet-wide topics (conventions, cross-project ADRs).
- **`projects/<slug>/`** — per-repo or per-effort wikis; each slug should have `_index.md`.
