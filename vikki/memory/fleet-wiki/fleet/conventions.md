# Fleet wiki conventions

- **Filenames:** `kebab-case.md` unless importing external titles.
- **Wikilinks:** Prefer `[[slug]]` to notes in the same tree; use relative paths for tools that do not resolve wikilinks.
- **Canon:** If Prism’s `shared.md` and this vault disagree, **update the vault** and trim `shared.md` after confirmation.
- **Spawns:** When Apex runs `/spawn-agent`, add `projects/<new-agent>/` or link from the MOC.
